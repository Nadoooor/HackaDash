# HackaDash
---------------------
An android application that has act as a dashboard for mentors in a track that fetch data from google sheet and uses appscript to run script function and sync with classroom.

## Configuration:
---------------------
1. Use these [google sheets](https://docs.google.com/spreadsheets/d/1A0_aaVznpAPzczByhRMdM-Pz52NpEHugwK7nPoqUc-0/edit?usp=sharing) and Appscript (You will find it inside the sheet) templates for your track.
2. Open the appscript code and click on the deploy on top bar.
3. create new and choose web app
4. make a new deployment and make it (Execute as: me), (Who can access: Anyone).
5. copy the deployment link (that is the link you will use in the app configuration).
6. Enjoy!! :)

## How to build:
--------------------------------------------
1. Download Android studio
2. git clone the repo and open it as a project in Android Studio.
3. Make whatever you want.
4. Build from the Main menu/build/Generate signed App Bundle or APK
5. fill in the info and make a new key. 
6. Choose it as APK not APP.
7. Enjoy!!!
